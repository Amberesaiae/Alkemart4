## Honest limits (SPA)

- Googlebot may not fully execute JS the same as SSR; **prerender/SSR for PDP** remains open.
- Static sitemap does **not** list every product — add a build job or API export later.
- Never invent ratings/reviews in schema; only API-backed fields.

## Dynamic sitemap (done)

```http
GET /store/sitemap              → application/xml
GET /store/sitemap?format=json  → JSON URL list
GET /store/sitemap?base=https://shop.example.com
```

Set `STOREFRONT_URL` or `SITE_URL` on the API for absolute `<loc>` hosts.
